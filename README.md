A personal portfolio site created for the repo owner to showcase his projects, built using React, SASS and Vite and deployed via Vercel.

<strong>Features:</strong><br>
  - Stylish, modern interface
  - React routing for internal links
  - Fully responsive
  - Optimised for the web
